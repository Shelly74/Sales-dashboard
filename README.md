# Sales-dashboard
provides real-time, interactive insights into key performance indicators (KPIs) such as total revenue, profit margins, sales growth, and regional performance.
It enables tracking of sales against targets, analysis of product and salesperson efficiency, and identification of trends through visuals like gauges, maps, and trend charts.
📊 1. Sales Dashboard Project
sales_data.csv → Dataset
analysis.py → Python analysis
👉 Use this file in Power BI to build:
Sales by region
Product performance
Time trends
🤖 2. Customer Churn Prediction
churn_data.csv → Dataset
model.py → Machine learning model
👉 Predict whether customers will churn
📄 3. Other Files
requirements.txt → Install dependencies
README.txt → Instructions

▶️ How to Use
Step 1: Install libraries
pip install -r requirements.txt

python analysis.py

Import .csv files
Create dashboards visually
